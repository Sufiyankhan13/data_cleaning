# Data Analysis Practice

A standard practice repository covering core Python data analysis libraries for beginners.

## What's Inside

- **NumPy** — array operations, mathematical functions, and numerical computing basics
- **Python Data Cleaning** — cleaning messy datasets, handling missing values, and transforming data using Pandas
- **Titanic** — exploratory data analysis and data manipulation practice using a real-world dataset
- **Visualization** — plotting and visual storytelling using Matplotlib and Seaborn

## Who is this for

Beginners who want hands-on practice with the most essential Python data analysis tools.

## Tools Used

- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
