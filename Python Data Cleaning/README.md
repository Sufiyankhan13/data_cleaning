# JSON Data Cleaning Pipeline — Pure Python

A data cleaning pipeline built with pure Python (no pandas).

## What it does
- Loads raw JSON ecommerce data
- Inspects for issues (whitespace, bad casing, nulls, invalid values)
- Cleans customer fields (name, email, age)
- Cleans item fields (price, quantity, category)
- Removes duplicate orders and items
- Saves cleaned output to new JSON file

## Skills demonstrated
- Pure Python data cleaning
- JSON parsing
- Data profiling and validation
- Pipeline design
